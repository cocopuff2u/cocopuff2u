My Story

I’m a Mac Administrator at NIWC Atlantic, a U.S. Navy organization that supports research, development, and secure technology systems. I have deep experience managing Apple and Linux systems across higher education, government, and high-compliance environments, with a focus on streamlining workflows, securing endpoints, and advancing open-source tooling to improve compliance and operational efficiency across macOS platforms.

I’m the lead developer of the open-source project M.A.C.E. (Mac Advanced Compliance Editor), the maintainer of MOFA (Microsoft Overview Feed for Apple), and an active contributor to the mSCP government working group, where I help drive improvements to macOS compliance frameworks and tooling. Through this work, I aim to make compliance more accessible, transparent, and practical for Mac administrators in high-security environments.

Beyond day-to-day development and operations, I’m especially passionate about systems engineering, endpoint security, and cross-platform automation, and I welcome collaboration in these areas.

Outside of work, I enjoy gaming, connecting with my community, and exploring Charleston, SC. I also have a deep love for outer space and hope to one day work with NASA or in a role that supports humanity’s exploration beyond Earth. It’s the nerd in me that yearns for the stars and believes they may one day be our next home.

Featured Projects

M.A.C.E.

Mac Advanced Compliance Editor
   

M.A.C.E. is a macOS app for Mac admins to easily create, document, and audit compliance baselines using NIST's mSCP 2.0 with visual rule editing and flexible management.

Check it out!

MOFA Website

   

I developed MOFA (Microsoft Overview Feed for Apple) using Vue.js and Python, hosted on GitHub Pages, to provide Mac administrators with automated, up-to-date feeds, tools, and direct download links for Microsoft Office apps on macOS.

Check it out!

BOFA Website

   

I built BOFA (Browser Overview Feed for Apple) with Vue.js and Python, hosted on GitHub Pages, to provide up-to-date browser feeds for Mac admins. It streamlines deployment with reliable XML, JSON, and YAML data.

Check it out!

DISA STIG Compliance Tool

 

I designed a DISA STIG compliance script using Bash in Visual Studio Code to automate the enforcement and verification of required security settings. The script checks and applies macOS STIGs to ensure systems meet DISA compliance standards.

Check it out!

Cocolabs Website

 

I built CoCoLabs using Hugo and hosted it on GitHub Pages as a central hub for all my projects. The site organizes and showcases tools and resources I’ve developed for Mac admins and the open-source community.

Check it out!
